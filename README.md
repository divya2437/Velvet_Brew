# ☕ Velvet Brew  
### *Slow Coffee. Soft Moments.*

A modern, aesthetic café website designed to deliver a calm, immersive digital experience — just like stepping into a cozy coffee corner in real life.

---

## ✨ Overview

**Velvet Brew** is a fully responsive café website built with a focus on:

- Smooth animations 🎬  
- Minimal, warm UI ☁️  
- Real-life usability (menu, cart, contact)

It blends design + functionality + storytelling — not just a website, but a vibe.

---

## 🚀 Features

### 🧭 Navigation & Layout
- Sticky glassmorphism navbar  
- Mobile-responsive hamburger menu  
- Smooth scrolling across sections  

### ☕ Interactive Menu
- Category-based filtering (Coffee, Desserts, Snacks)  
- Dynamic rendering using JavaScript  
- Add-to-cart functionality  

### 🛒 Smart Cart System
- Sidebar cart UI  
- Quantity tracking  
- Total price calculation  
- WhatsApp order integration  

### 🎬 Animations & UX
- Scroll reveal animations (Intersection Observer)  
- Staggered menu card animations  
- Toast notifications (micro-interactions)  
- Smooth transitions & hover effects  

### 🖼️ Visual Sections
- Hero section with animated zoom background  
- Gallery grid with hover zoom  
- Testimonials slider (auto + manual)  
- Cinematic “About” section  

---

## 🎯 Special Highlight

### 🎥 “A Corner for the Soul” Section

- Image → slide in from left  
- Text → fade in from right  

Creates a **dual-direction motion effect** for a premium feel.

---

## 🛠️ Tech Stack

- HTML5 – Structure  
- CSS3 – Styling & animations  
- JavaScript (Vanilla) – Logic & interactivity  

> No frameworks. No shortcuts. Just pure control 💅
